今回はゲームのプロジェクトと企画書、仕様書、ゲーム内で使用するリソースを全てここで管理したい!
一応,ある程度階層は作っておいたから基本はそれに従ってほしい.

"ゲームのプロジェクト"は"ゲームプロジェクト"フォルダ以下にて管理
"企画書、仕様書、ゲーム内で使用するリソース"は"資料"フォルダにて管理するとする

あと"最新版として起動できるもの"が完成したらそれを"ゲーム"フォルダ以下にて管理する
"ゲーム"フォルダに入れるものはリリースビルドでビルドした実行ファイルとリソースのみにする
